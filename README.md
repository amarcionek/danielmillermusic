# Daniel Miller & The High Life — Official Website

Static website for Daniel Miller, a Tennessee-born Americana/roots singer-songwriter now based near Boston, MA. The site serves as his public-facing presence for music, tour dates, and booking inquiries.

## Structure

- [index.html](index.html) — single-page site with sections for Hero, News, About, Music, Tour Dates, and Contact
- [styles.css](styles.css) — all styling; dark Americana aesthetic with a warm rust/ember color palette
- [images/](images/) — hero photos, album art, and single covers

## Sections

- **Hero** — auto-advancing photo slideshow with links to stream the latest single
- **News** — latest release announcements with streaming links
- **About** — artist bio
- **Music** — discography shelf (singles and albums, 2011–present)
- **Tour** — upcoming show dates with venue, time, and solo/full-band indicator
- **Contact** — booking email and social media links (Facebook, Instagram, Twitter/X)

## Tech

Plain HTML/CSS/JS — no build tools or dependencies. Fonts loaded from Google Fonts.
